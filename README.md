MovieMaster Pro

It is a complete movie management web application that allows users to browse,add,edit, delete and manage their own movie collections and watchlist. It includes authentication, protected routes, full CRUD operations, loading states, toast notifications and fully responsive design.

Key Features
1.Dynamic featured carousel
2.Full CRUD support
3.Protected routes
4.Top rated and recent movies
5.My watchlist
6.Theme toggle
7.Loading and toast notifications
8.Advanced filtering
